# Case 08: Coagulation

Interactive educational website for pathology residents.

## Topic

The bleeding patient with a lupus anticoagulant: assay interference or real disease?

## Contents

- `index.html` - main static website
- `styles.css` - responsive academic styling
- `script.js` - sidebar navigation, case questions, answer explanations, and score tracking
- `.nojekyll` - GitHub Pages compatibility file

## How to Use

Open `index.html` in a browser, or upload this folder to GitHub and enable GitHub Pages.

No build tools, package managers, or external dependencies are required.
