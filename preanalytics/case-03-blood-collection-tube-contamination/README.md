# Case 3: Blood Collection Tube Contamination

Interactive educational module for pathology residents learning about preanalytical laboratory errors related to blood collection tube contamination.

## GitHub Pages Files

- `index.html` - main educational module
- `styles.css` - responsive styling
- `script.js` - navigation, quiz feedback, and score tracking
- `assets/` - reserved for future images or downloadable teaching assets

## Deploy to GitHub Pages

1. Upload these files to a GitHub repository.
2. In the repository settings, open **Pages**.
3. Set the source to the repository branch containing `index.html`.
4. Save and wait for GitHub Pages to publish the site.

No build step, dependencies, or external services are required.
