# Case 4: Inappropriate Blood Collection Tube Volume

Interactive educational module for pathology residents covering inappropriate blood collection tube volume as a preanalytical laboratory error.

## GitHub Pages Deployment

Upload the contents of this folder to a GitHub repository and enable GitHub Pages from the repository settings.

The site entry point is:

- `index.html`

## Files

- `index.html` - single-page educational module
- `styles.css` - responsive academic styling
- `script.js` - navigation, progress tracking, quiz scoring, and interactive clinical cases
- `assets/` - reserved folder for future image or media assets

## Local Use

Open `index.html` in a browser. No build step, server, framework, or internet connection is required.
