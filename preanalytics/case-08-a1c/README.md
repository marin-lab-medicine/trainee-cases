# Case 8: A1c

Interactive educational module for pathology residents covering HbA1c weighting, erythrocyte lifespan, biologic and analytic interference, hemoglobin variants, clinical case reasoning, and knowledge-check questions.

## GitHub Pages Deployment

This folder is ready to upload to a GitHub repository and serve with GitHub Pages.

## Files

- `index.html` - single-page educational module
- `styles.css` - responsive academic layout and visual styling
- `script.js` - progress tracking, tabbed content, branching cases, reveal controls, mechanism explorer, and quiz feedback
- `assets/images/` - image assets used by the module

## Local Preview

Open `index.html` directly in a browser, or serve the folder with a static web server.
