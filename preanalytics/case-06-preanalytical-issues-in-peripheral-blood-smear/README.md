# Case 6: Preanalytical Issues in Peripheral Blood Smears and CBCs

Static interactive educational module for pathology residents.

## Files

- `index.html` - module content and structure
- `styles.css` - responsive academic visual design
- `script.js` - navigation, progress tracking, branching cases, and quiz scoring
- `assets/` - teaching images used in the clinical cases

## GitHub Pages Deployment

Upload the contents of this folder to a GitHub repository. In the repository settings, enable GitHub Pages and choose the branch/folder containing `index.html`.

All links are relative, so the site can run from the repository root or from a GitHub Pages project path.
