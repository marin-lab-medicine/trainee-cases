# Case 7: Synovial Fluid

GitHub Pages-ready educational module for pathology residents covering synovial fluid preanalytics, specimen handling, crystal mimics, microscopy pitfalls, interactive clinical cases, and knowledge-check questions.

## Project Structure

- `index.html` - main single-page educational module
- `styles.css` - responsive academic styling
- `script.js` - progress tracking, tabs, branching cases, and quiz feedback
- `assets/` - reserved for future images or media

## GitHub Pages Deployment

Upload the contents of this folder to a GitHub repository, then enable GitHub Pages from the repository settings. The site uses only relative links and has no framework or build step.
