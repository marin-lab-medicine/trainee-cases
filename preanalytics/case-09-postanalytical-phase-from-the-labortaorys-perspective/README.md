# Case 9: Postanalytical Phase from the Laboratory’s Perspective

Interactive educational module for pathology residents, built with plain HTML, CSS, and JavaScript.

## Open locally

Open `index.html` directly in a modern browser, or serve this folder with any static web server.

## Publish with GitHub Pages

1. Create a GitHub repository and upload the contents of this folder to the repository root.
2. In the repository, open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and the `/ (root)` folder, then save.

All website paths are relative, so the module works from a GitHub Pages project URL as well as from a local extracted folder.

## Features

- Nine educational sections with responsive sidebar navigation
- Browser-based progress persistence using `localStorage`
- Interactive result-release workflows
- Five branching clinical cases with immediate feedback
- Eight-question scored knowledge check with explanations and retry
- Mobile navigation, keyboard focus states, and reduced-motion support
