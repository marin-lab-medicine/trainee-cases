# Case 5: Lab Mitigation Strategies for Preanalytics

Interactive educational module for pathology residents covering laboratory mitigation strategies for preanalytical and analytical quality risks.

## Contents

- `index.html` - main educational module
- `styles.css` - responsive academic styling
- `script.js` - navigation, progress tracking, quiz feedback, and interactive cases
- `assets/` - module images

## GitHub Pages Deployment

1. Upload the contents of this folder to a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Select the branch containing these files.
4. Choose the repository root as the publishing source.
5. Save and open the generated GitHub Pages URL.

## Local Preview

Open `index.html` in a browser. The module uses only HTML, CSS, and JavaScript, so no build step or framework is required.
